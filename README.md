#JDesktop

Easily set a custom Desktop on your school laptop with JDesktop.

###Get the code

Download the latest source code with:

    git clone git://github.com/bobbungee/JDesktop.git
	
Keep your source up to date with:

    git pull origin master

Subversion is also supported:

    svn checkout http://svn.github.com/bobbungee/JDesktop.git

### Dependencies

 * [Java Development Kit](http://www.oracle.com/technetwork/java/javase/downloads/)
